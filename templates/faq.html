Home Faqs

1.What is Rapsberry Pi | Web API?

Raspberry Pi | Web API gives you opportunity to record, stream, make screenshots, watch recorded videos and captured screenshots from Website, delete them or convert recorded .h264 videos to .mp4 using FFmpeg which must be compiled yourself.

2.Supported Modules

At the moment this API supported standard camera module for Raspberry Pi and B101 HDMI to CSI-2 Bridge (15 pin FPC) from Auvidea GmbH.

3.Recording

Raspberry Pi recording videos in .h264 output format, you can open them with VLC Player. In order, you can convert them in .mp4 format using Web API. If you want to convert them in other format, please download converter and use it.

4.Streaming

At the moment only udp-streaming thougth FFmpeg is avaible. We are working at the solution which will give you opportunity to stream in udp, rtp and rtsp without using FFmpeg. If you make some screenshots at streaming, they will not be saved in recording-%date% directory

5.Settings

At the moment you can only change settings like, encoder settings and date settings. We are working at the solution which will give you opportunity to change network settings, path where videos and screenshots will be saved, quality of recorded videos and screenshots and more.

6.Release?

This is pre-beta release. Final release will come out soon.
